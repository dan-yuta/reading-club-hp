# 読書会HP用 画像生成プロンプト集

## 共通スタイル指示（全画像に追記してください）

```
スタイル: フラットイラスト、温かみのある水彩風タッチ、パステルトーン
配色: 青系（ネイビー、スカイブルー、ライトブルー）をメインに、アクセントでオレンジ・グリーン
人物: 20〜30代の日本人男女、カジュアルな服装、親しみやすい雰囲気
背景: シンプル・白背景または淡い色のグラデーション
サイズ: 横長（16:9）で出力してください
テイスト: Airbnb、MUJI、Notion のようなクリーンで洗練された雰囲気
重要: 画像内に日本語・英語を含む一切の文字やテキストを描画しないでください。吹き出し、ラベル、看板、画面上の文字なども含めず、純粋なイラストのみにしてください。
重要: ノートパソコンは人物の腰〜胸あたりの高さに収まる程度の大きさで描いてください。人物より大きく描かないこと。人物の顔や表情が主役です。
```

## 画像1: ヒーロー画像（hero.jpg）
**配置場所:** ページ最上部のメインビジュアル右側
**用途:** 読書会の世界観を一目で伝えるキービジュアル

```
オンラインZoom読書会のイメージイラスト。
手前に一人の若い女性がノートパソコンの前に座り、画面越しに笑顔で話している。
ノートパソコンの画面には、小さなビデオ通話のタイル状の枠が3〜4個並び、それぞれ笑顔の参加者の顔が映っている。
手前の女性の横には本とコーヒーカップが置かれている。
ノートパソコンは女性の上半身より小さく、テーブルの上に自然に置かれたサイズで描く。
全体的に温かみがあり、「一人でも気軽に参加できる」安心感が伝わる構図。
文字やテキストは一切含めないこと。
```

## 画像2: 安心ポイント用（reassurance.jpg）
**配置場所:** 「安心ポイント」セクションの背景または上部
**用途:** 初参加でも安心できる雰囲気を伝える

```
Zoomの読書会に初めて参加する若い女性が、自宅のデスクでノートパソコンの前に座っているイラスト。
少し緊張しながらも笑顔で画面に向かって手を振っている。
ノートパソコンの画面には、手を振り返している他の参加者の小さな顔が映っている。
参加者同士の間にハートマークや星マークなどの温かいアイコンが浮かんでいる。
ノートパソコンはテーブルの上に自然なサイズで置かれ、人物の顔と表情がメインに見える構図。
全体に柔らかい光が差し込み、安心感と温かさを表現。
文字やテキスト、吹き出し内の文字は一切含めないこと。
```

## 画像3: 成長できる理由用（growth.jpg）
**配置場所:** 「成長できる3つの理由」セクションの上部
**用途:** アウトプット・視野の広がり・仲間の3要素を視覚化

```
横長の画像を均等に3分割し、左・中央・右にそれぞれ1つずつ独立したシーンを描いたイラスト。
3つのシーンは薄いグレーの縦線で区切る。背景色は3シーンとも統一して淡いスカイブルーのグラデーション。

【左のシーン】
20代の日本人男性が1人、自宅のデスクに座っている。
テーブルの上に小さなノートパソコンが開いてあり、Zoomのビデオ通話中。
男性は右手に開いた本を持ち、左手を前に出してジェスチャーしながらカメラに向かって話している。
男性の頭の右上に黄色い電球のアイコンが1つ浮かんでいる。

【中央のシーン】
ノートパソコンの画面を正面から見た構図。
画面の中にZoomのビデオ通話が映っており、左側に20代の日本人女性、右側に20代の日本人男性の顔が大きく映っている。
2人ともそれぞれ本を手に持ち、笑顔で向き合うように話している。
画面の上部に、オレンジ色と水色の2つの小さな光の輪が重なり合っている。

【右のシーン】
ノートパソコンの画面を正面から見た構図。
画面の中にZoomのギャラリービューが映っており、4人の20〜30代の日本人男女の顔が2×2のグリッドで並んでいる。
4人全員が笑顔で、それぞれ手に本を持って画面に向かって掲げている。

文字やテキストは一切含めないこと。画面上のUI文字も不要。
```

## 画像4: 発表の流れ用（flow.jpg）
**配置場所:** 「発表の流れ」セクションの上部
**用途:** Zoomでの読書会のリアルな雰囲気を伝える

```
Zoomで読書会の発表をしている様子のイラスト。
中央に一人の参加者が自宅のデスクで、ノートパソコンの前に座り、本を手に持って画面に見せながら発表している。
ノートパソコンの画面には、うなずきながら聞いている他の参加者の顔が3〜4人分、小さなタイルで映っている。
発表者の表情は自信に満ちていて、画面の中の参加者も真剣かつ楽しそう。
ノートパソコンは発表者の上半身より小さく、テーブルに自然に置かれたサイズで描く。人物と本が主役。
文字やテキスト、チャット欄の文字は一切含めないこと。
```

## 画像5: CTA用（cta.jpg）
**配置場所:** ページ下部の申込みCTAセクション付近
**用途:** 参加を後押しする最後のひと押し

```
自宅のデスクでノートパソコンを開き、Zoom読書会に参加しようとしている若い男女2人のイラスト（左右に分かれた分割画面構成）。
それぞれが本を片手に、もう片方の手でパソコンに向かって笑顔で手を振っている。
背景にはそれぞれの部屋に本棚やコーヒーカップ、観葉植物が見える。
ノートパソコンはテーブルの上に控えめなサイズで描き、人物の笑顔がメイン。
色調は明るく爽やかで、「これから楽しいことが始まる」ポジティブな印象。
文字やテキストは一切含めないこと。
```

## 画像生成後の手順

1. 生成した画像を以下のファイル名で保存:
   - `hero.jpg` → `assets/images/hero.jpg`（既存を上書き）
   - `reassurance.jpg` → `assets/images/reassurance.jpg`
   - `growth.jpg` → `assets/images/growth.jpg`
   - `flow.jpg` → `assets/images/flow.jpg`
   - `cta.jpg` → `assets/images/cta.jpg`

2. 画像をHP関連フォルダの `assets/images/` に配置

3. Claude Codeに「画像を配置して」と依頼すると、HTML/CSS/JSに組み込みます
